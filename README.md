# Kerala Migrant Health - Digital Health Management System

A comprehensive digital health management system designed specifically for migrant workers in Kerala, India, aligned with UN Sustainable Development Goals (SDGs).

## üåü Features

### Core Health Management
- **Health Registration**: Easy profile creation and management for migrant workers
- **Health Tracking**: Monitor vital signs, medications, and symptoms
- **Emergency Services**: Quick access to emergency healthcare and SOS features
- **Health Education**: Preventive care, nutrition, and wellness modules
- **Multilingual Support**: Available in English, Malayalam, Hindi, and Tamil

### SDG Alignment
This system directly supports:
- **SDG 3**: Good Health and Well-being
- **SDG 10**: Reduced Inequalities
- **SDG 17**: Partnerships for the Goals

### Technical Features
- **Progressive Web App (PWA)**: Works offline and installable on mobile devices
- **Responsive Design**: Optimized for mobile and desktop
- **Accessibility**: WCAG compliant with screen reader support
- **Data Privacy**: Local storage with optional cloud sync
- **Real-time Notifications**: Health reminders and emergency alerts

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for initial setup (offline mode available after)

### Installation
1. Clone or download the repository
2. Open `index.html` in a web browser
3. For PWA installation:
   - Click the "Install" button when prompted
   - Or use browser menu: "Add to Home Screen"

### Usage
1. **Registration**: Click "Register Now" to create your health profile
2. **Health Tracking**: Use the dashboard to log vitals, medications, and symptoms
3. **Emergency**: Access emergency services and find nearest hospitals
4. **Education**: Explore health education modules
5. **Profile**: Manage your personal information and view health statistics
